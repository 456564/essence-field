# 物理本质空间 — 8物理算子视觉架构

## 架构转向（2026-05-30）

旧八卦语义算子（乾=圆、坤=平...）是人为解释，不是物理测量。新架构：8 算子测量可验证的物理量，理解从物理推演产生。

## 8 物理算子依赖链

```
动(梯度) ── RGB 直接可测
静(无变化) ── 动的互补
  │
  ├→ 刚(硬边界) ── 需要动定位边缘
  ├→ 柔(软纹理) ── 需要动+静区分纹理/边界
  │     │
  │     ├→ 聚(围合) ── 需要刚判定"被包围"
  │     ├→ 散(开放) ── 不被包围的暴露区域
  │           │
  │           ├→ 阳(实体) ── 需要聚
  │           ├→ 阴(虚空) ── 聚的补集
```

## 流水线

```
RGB [B,3,H,W]
  ↓ PhysicalOperatorLayer (8算子)
8 响应图 [B,8,H,W]
  ↓ 1×1 conv 投影 (1→8 per operator)
8×8 特征 [B,8,8,H,W]
  ↓ reshape
64 维本质场 [B,64,H,W]
```

## 设计约束

- 所有算子输出非负
- 投影权重 clamp ≥ 0
- 下层不依赖上层（动不读刚）

## 设计原则

1. **每次只实现一个算子** — 做完验证再做下一个
2. **依赖链顺序** — 上层用的下层必须先存在
3. **算子 = 纯函数** — 输入 tensor，输出 tensor，无状态
