# 八卦→64卦 视觉架构

## 核心数据结构

**64 维卦象场 [B, 64, H, W]**

- 每个像素携带 64 维向量 = 该位置"物质本质"的描述
- 同一物质的像素 → 64 维向量相似
- 不同物质的像素 → 64 维向量不同
- 64 维向量不是图片级别的描述，是像素级别的描述

**不要取全图均值。取均值 = 丢弃空间信息 = 背离架构核心。**

## 8 算子（固定数学运算）

| 卦 | 算子 | 检测目标 |
|:--|:----|:--------|
| 乾 | 圆对称性 | 圆形/全局结构 |
| 坤 | 局部平坦度 | 平坦/背景/承载面 |
| 震 | 拉普拉斯 | 边缘/突变 |
| 巽 | 方向梯度一致性 | 细长纹理/方向性蔓延 |
| 坎 | 等照度线曲率 | 弯曲/流动 |
| 离 | 梯度幅值×色彩饱和度 | 视觉能量/颜色/光照 |
| 艮 | 局部纹理不连续性 | 块状/遮挡边界 |
| 兑 | 中心-周围对比度 | 凹陷/缺口 |

每个算子输出 [B, 1, H, W]。

## 双线性融合

```
8 算子响应 [B, 1, H, W] × 8
    ↓ 1×1 conv 投影到 8 维
8×8 特征矩阵 [B, 8, 8, H, W]
    ↓ 可学习双线性核 A ∈ R^(8×8)
64 维卦象场 [B, 64, H, W]
```

## 64 维本质空间的两层信息（已验证）

每个 64 维向量包含两层独立信息：

### 第一层：向量范数 → 物体位置（活跃度）
- 背景范数：1.8~2.2，低且均匀
- 物体范数：2.2~3.0+，高
- 两者完全分离（双峰分布），可无监督分割物体
- 范数与表面曲率正相关：平坦→低，边缘/高光→高
- 不区分亮暗：阴影和亮面同等地被判定为物体

### 第二层：向量方向 → 物质类型
归一化后（去掉范数），方向编码物质身份：
- 亮面：一种方向签名
- 阴影：另一种方向签名
- 边缘：与前两者差异最大的签名
- 同种物质在不同位置的方向签名一致

在杯子上验证，K=3 方向聚类稳定分出：亮面、阴影、边缘。
