# 路线图：从"证明场好用"到"证明场有用且可解释"

## 当前状态

固定颜色版 (v5) 已达到阶段性高峰：
- 100% 通过率，平均分离度 15.6
- 背景纯黑无噪点，物体内部连续无空洞
- 算子完全固定，无颜色可训练参数

分离度只是单图自监督产物，不能证明64维场具有通用视觉表征能力。

---

## 阶段一：表征质量加固（当前→3周内）

**目标**：确保64维场在不同数据、不同条件下都是稳定且鲁棒的底层特征。

### 任务

- [ ] **全量数据训练**：Caltech101全部9144张图，epoch 50+
  - 内部平滑损失：`L_smooth = mean(|norm_i - norm_j|)` 对物体掩码内相邻像素
  - 背景零化损失：`L_bg = mean(norm[bg])` 鼓励背景范数趋近零
  - 学习率余弦退火
- [ ] **增广鲁棒性测试**：亮度(0.5×,1.5×)、高斯噪声(σ=0.05)、JPEG(质量30)
  - 分离度下降不超过30%

### 验收指标

| 指标 | 目标 |
|:----|:----|
| 分离度 > 2.0 比例 | > 98% |
| 物体区域范数CV | < 0.25 |
| 背景误检率 | < 1% |
| 扰动后分离度保持率 | > 70% |

---

## 阶段二：下游任务验证（2~4周）

**目标**：证明64维场包含语义信息，可作为通用视觉特征。

### 任务

- [ ] **线性分类探测**：全局平均池化64维场 → 线性SVM/逻辑回归(64→101类)
  - 对比基准：随机权重、ResNet-18第一个conv块
- [ ] **显著性检测**：直接在DUTS-TE上以范数作为显著图
  - 计算MAE和F-measure，对比IT、AIM等经典算法
  - 不需要训练，直接推理

### 验收指标

| 指标 | 目标 |
|:----|:----|
| 线性分类Top-1准确率 | > 30%（101类随机1%） |
| 显著性检测MAE | < 0.2 |
| 显著性检测F-measure | > 0.5 |

---

## 阶段三：可解释性与理论构建（同步进行）

**目标**：揭示64维场中卦象的组合规律，让模型透明。

### 任务

- [ ] **投影层权重分析**：可视化 `|W_proj|` (8×8)，看算子被强化/抑制关系
- [ ] **颜色响应验证**：纯色块输入，验证每个算子响应是否匹配卦象颜色
- [ ] **A核谱分解**：特征值分解8×8 A核，观察主方向对应的卦象对
- [ ] **生成可解释文档**："64维场的第一主成分主要由离火+震雷构成"

### 验收指标

- 算子颜色响应雷达图与八卦颜色匹配度 > 70%
- 可解释文档：64维场主成分的卦象组合分析
