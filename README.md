# 八卦→64卦 视觉架构

> **物理先验驱动的深度解耦表示学习框架。以八卦作为组织原则，目标是让模型学会"感知世界的本质属性"而非"记忆物体标签"。**

---

## 一、核心哲学

世界任何物质都可以用 **8 种基本"象"** 的组合来描述。

- 8 象两两交互 → 64 种关系 = 物质在"本质空间"中的唯一坐标
- 不用分类标签，不用黑盒特征向量
- 目标：模拟世界本质，实现**抗遮挡、可解释、组合泛化**的视觉理解

---

## 二、架构总览

```
输入图像 (2D)
   │
   ▼
8个异构子网络（乾、坤、震、巽、坎、离、艮、兑）
   │ 每个按该卦的"象"从底层结构设计，架构上不可通约
   │ 输出：8张空间对齐的特征图
   ▼
位置外积 → 64维卦象场
   │ 每个像素得到一个64维向量，即其物质本质描述
   ▼
基本单元 = 空间点 + 64维向量（世界的最小不可分单位）
   ▼
本质空间聚类/组合 → 物体实例 + 物质签名
   ▼
下游任务：识别、分割、预测、闭环
```

---

## 三、8 子网络异构设计

每个子网络通过**硬结构约束 + 信息瓶颈**锁定感知偏好，天然只响应一种"象"，从源头防止特征崩塌。

| 卦 | 核心象 | 感知目标 | 关键设计 |
|:--|:------|:--------|:--------|
| **乾☰** | 刚健、圆、主导 | 全局对称结构、圆形、视觉主导性 | 大核膨胀卷积、全局上下文、旋转等变 |
| **坤☷** | 柔顺、平坦、承载 | 大面积平坦、均匀背景 | 每层后强制大核均值池化 + 通道压缩瓶颈 |
| **震☳** | 动、突变、爆发 | 高频边缘、剧变 | 固定 Sobel/Laplacian 前置 + 硬阈值截断 |
| **巽☴** | 入、散、渗透 | 细长纹理、方向性蔓延 | 多方向 Gabor/条形卷积，小核限制 |
| **坎☵** | 陷、险、弯曲流动 | 凹陷、曲线、流形 | 可变形卷积、曲率感知 |
| **离☲** | 明、附着、热烈 | 光照模式、视觉显著、能量感 | 1×1 卷积 + InstanceNorm + 亮度/色度双分支 |
| **艮☶** | 止、阻隔、块状 | 大块实体、遮挡边界 | 大核深度可分离卷积 + 遮挡检测辅助 |
| **兑☱** | 毁缺、反射、缺口 | 局部异常、镜面高光、裂纹 | 局部对比度异常检测（原图-平滑） |

## 四、子网络学习目标

不是记忆"样本"，而是学习 **"象的本质不变量"** ，获得从任意图像推理卦象强度的能力。

> **例：** 离网络学会的不是"火=离"，而是：
> 任何呈现"高亮度、视觉显著、暖燥热烈、能量放射"特征的区域，无论火、灯泡、夕阳、LED，都能输出高离象响应。同时排除物体形状（乾负责）、纹理（巽负责）、材质（艮/兑负责）的干扰。

---

## 五、64维本质空间与基本单元

- **64维向量**：每个像素的完整物质描述，64个维度各有明确的物理含义
  - 对角元（乾×乾）：某象的纯度
  - 非对角元（乾×兑）：两象的交互（如"圆形上的缺口"）
- **基本单元**：空间中的一个点，携带64维卦象向量，是世界最小不可分单位
- **物质识别**：同一物质的像素在64维空间中自然聚类，不同物质分离
- **物体形成**：本质空间相似 + 空间邻近的基本单元聚合

## 六、训练策略

| 阶段 | 内容 | 目的 |
|:----|:----|:----|
| 阶段1 | 纯合成单一象数据，各子网络独立预训练 | 学会提取纯粹的"象" |
| 阶段2 | 混合双象合成数据，冻结子网络，训练融合层 | 学会处理象的交互 |
| 阶段3 | 真实数据微调融合层 + Adapter，子网络极低学习率 | 迁移到真实世界，防止坍塌 |

**防崩塌机制：** 硬结构约束 + 信息瓶颈 + Adapter 残差校准（不修改子网络本身）

---

## 七、关键验证指标

| 指标 | 含义 |
|:----|:----|
| **解耦度** | 子网络特征互信息、线性分类可分离性、子网络替换实验（性能应骤降） |
| **物理一致性** | 同材质在不同光照/角度下 64 维向量的类内/类间方差比 |
| **组合泛化** | 见过"圆形"+"缺口"，能否零样本识别"带缺口的圆盘" |
| **抗遮挡** | 遮挡后剩余像素的卦象分布与原始分布的 KL 散度 |
